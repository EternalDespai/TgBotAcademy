MyBotAcademy — Telegram бот для отчётов из Excel (.xlsx)
Telegram-бот принимает .xlsx файлы и формирует отчёты по нескольким шаблонам (расписание, темы уроков, отчёт по студентам, посещаемость преподавателей). Проект рассчитан на Python 3.12.

Структура проекта

MyBotAcademy/
├─ bot_app/
│  ├─ __init__.py
│  ├─ config.py          # BOT_TOKEN
│  ├─ excel_parser.py    # логика анализа Excel + генерация отчётов
│  ├─ main.py            # запуск Telegram-бота
│  └─ utils.py           # send_long_message (длинные сообщения)
├─ requirements.txt      # зависимости
├─ start_bot.bat         # запуск без IDE (Windows)
└─ README.md

Возможности бота
Бот автоматически определяет тип Excel-файла и запускает подходящий метод:

Метод 1 — Расписание

Считает количество пар по предметам (ищет строки, где встречается Предмет:).

Метод 2 — Темы уроков

Проверяет корректность “Темы урока” по шаблону:
Урок № <число>. Тема: <текст>
Ошибки группируются по предметам.

Метод 3 — Отчёт по студентам
Находит студентов с:

ДЗ = 1 (в коде это <= 1.05)
КР < 3
Ожидаются колонки: FIO, Homework, Classroom.

Метод 4 — Посещаемость по преподавателям

Выводит преподавателей, у кого средняя посещаемость ≤ 40%.
Ожидаются колонки: ФИО преподавателя и Средняя посещаемость.
Результат возвращается в Telegram. Если текст большой — сообщение автоматически разбивается на части (см. utils.send_long_message).

Требования:

Python 3.12

Windows / Linux / macOS (скрипт start_bot.bat — только Windows)

Доступ в интернет (для установки зависимостей) и доступ Telegram.

Установка через start_bot.bat (Windows)

Установи Python 3.12.

Открой папку проекта.

Настройка токена
Токен хранится в bot_app/config.py.

Пример содержимого: BOT_TOKEN = "ВАШ_ТОКЕН_ОТ_BOTFATHER"

Запусти start_bot.bat.

Скрипт создаст виртуальное окружение venv, установит зависимости из requirements.txt и запустит бота.
